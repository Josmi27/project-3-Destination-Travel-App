WHAT HAVE WE MADE? WHAT DOES IT EVEN DO?
  • We have made a real-time chat app accessible from the browser. The chat app
    features an international chatbot which enables users to do tasks like: find local
    airports to various countries, provide users with up to date travel advice, and
    provide recommended songs for the selected destination

HOW DID WE MAKE IT? (Technologies Included)
  • To create this chat app, we utilized AWS Cloud9 as environment to test and create
    our code. We used Flask for the backend of our site and React/JS for the front-end,
    tied in with SocketIO. To expand on the features of our application we pulled in
    multiple APIs: Genius Music, Currency Converter-5, My Memory API, Cometari
    Airport Finder, Reisewarnung API, and Trail API. These APIs allowed us to access
    music, airport names, travel advisories, currency conversions, activities and more.
    We stored our code on a group repository on Github.

WHY DOES IT MATTER?
  • As Computer Science students heading into industry in the coming months, it was
    important that we understood all that need to go into creating a complex application.
    When creating a project, it is important to know who and what you are are making
    it for to make sure that all possible needs and wants are taken care of. Creating this
    chat app, with a theme focused on travel, helped us learn how to manuever around
    the "wants" of the customer and to put out a product that worked and would make
    people happy. Using this travel theme allowed us to think about what people would
    actually want to see in an application and what features needed to to create a great
    app. This project helped us experience a lot of what we will be experiencing as we
    enter into industry.
